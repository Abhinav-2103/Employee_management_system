# 🧑‍💼 Employee Management System

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![GUI](https://img.shields.io/badge/GUI-CustomTkinter-green)
![Database](https://img.shields.io/badge/Database-MySQL-orange)

A **GUI-based Employee Management System** built with **Python and CustomTkinter**.  
The application allows users to manage employee records including adding, updating, deleting, and viewing employee details.

---

# 📌 Project Overview

This system helps organizations manage employee information efficiently through a graphical interface.

It demonstrates:

- GUI development using **CustomTkinter**
- Database connectivity with **MySQL**
- CRUD operations
- Data management using Python

---

# 🚀 Features

✅ Add new employees  
✅ Update employee details  
✅ Delete employee records  
✅ View employee information  
✅ Database integration with MySQL  
✅ GUI interface using CustomTkinter
